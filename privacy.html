1. 개인정보의 수집 / Data Collection

본 앱은 회원가입, 로그인 기능 없이 오프라인 학습만 제공하며 개인정보를 직접 수집하지 않습니다.

This application does not collect any personal information, as it provides offline learning features without sign-up or login.

2. 개인정보의 이용 목적 / Purpose of Data Use

앱은 서버로 어떠한 개인정보도 전송하지 않으며, 학습 진행도 등 데이터는 사용자 기기에만 저장됩니다.

No data is transmitted to external servers; learning progress and responses are stored locally on the user’s device only.

3. 제3자 제공 및 위탁 / Third-Party Sharing

본 앱은 수집된 개인정보가 없으므로 제3자 제공이나 위탁 처리를 하지 않습니다.

Since no personal information is collected, no data is shared with or entrusted to any third party.

4. 보유 및 파기 / Data Retention and Deletion

앱은 서버에 정보를 보관하지 않으며, 사용자는 앱 삭제 또는 설정 초기화를 통해 로컬 데이터를 제거할 수 있습니다.

No data is stored on servers. Users can delete their local data by uninstalling the app or using in-app reset features.

5. 아동의 개인정보 / Children’s Privacy

본 앱은 아동의 개인정보를 수집하지 않으며, 보호자 지도하의 사용을 권장합니다.

This app does not knowingly collect personal data from children and encourages supervised use by guardians.

6. 국제 전송 / International Data Transfer

개인정보를 서버로 전송하지 않으므로 국외 이전이 발생하지 않습니다.

No personal data is transmitted internationally since no remote storage is used.

7. 이용자 권리 / User Rights

개인정보가 저장되지 않으므로 열람·정정·삭제 요청은 적용되지 않습니다. 데이터 삭제는 기기 설정을 통해 직접 수행할 수 있습니다.

As no data is stored, requests for access, correction, or deletion do not apply. Users may delete data directly via device settings.

8. 보안 조치 / Security Measures

앱은 최소한의 권한만을 사용하며, 네트워크 통신 없이 로컬에서 실행됩니다.

The app operates offline with minimal permissions and without external network communication to ensure data safety.
