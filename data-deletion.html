사용자 데이터 삭제 안내

본 페이지는 Polyglot, Inc. 서비스(이하 “서비스”) 이용자가 자신의 데이터를 삭제하는 방법과, Facebook “데이터 삭제 안내 URL” 요구사항을 충족하기 위해 제공됩니다.

요약

  • 앱 내에서 직접 삭제: 권장 · 설정 → 계정 > 계정 삭제
  • 웹 양식으로 요청: 삭제 요청 양식 (신원 확인 필요)
  • 이메일로 요청: privacy@polyglot-inc.com (또는 귀사 전담 이메일)
  • Facebook 전용: Facebook 데이터 삭제 지침 제공

앱/웹에서 직접 삭제하기

  1. 서비스에 로그인하십시오.
  2. 설정 → 계정 메뉴로 이동하십시오.
  3. 계정 삭제 버튼을 선택하고, 안내에 따라 본인 확인(비밀번호/2차 인증)을 완료하십시오.
  4. 삭제가 접수되면 계정은 비활성화 상태가 되며, 아래 처리 기한 내에 완전 삭제가 진행됩니다.

참고 결제/세금 관련 법적 보존 항목은 즉시 삭제가 제한될 수 있습니다.

양식/이메일로 삭제 요청하기

① 삭제 요청 양식 (예시 HTML)

<form method="POST" action="https://polyglot-inc.com/api/v1/privacy/delete-request">
  <label>이메일<input type="email" name="email" required></label>
  <label>서비스 계정 ID(선택)<input name="user_id"></label>
  <label>요청 사유(선택)<textarea name="reason"></textarea></label>
  <button type="submit">삭제 요청</button>
</form>

② 이메일

privacy@polyglot-inc.com 으로 아래 정보를 보내 주십시오.

  • 이메일 주소(로그인 계정)
  • 성명(또는 닉네임)
  • 계정 ID(알고 계신 경우)

본인 확인을 위해 추가 정보를 요청드릴 수 있습니다.

Facebook 데이터 삭제 안내

Facebook 로그인으로 서비스에 가입하신 경우, 아래 두 방법 중 하나로 삭제를 요청하실 수 있습니다.

방법 A — 직접 삭제(권장)

  1. 서비스에 로그인 → 설정 → 계정 → 계정 삭제 진행

방법 B — Facebook 요청 URL

아래 전용 엔드포인트로 접속하시면 Facebook 계정 식별자(FBID) 기반의 삭제 접수가 이루어집니다.

https://polyglot-inc.com/facebook/data-deletion?fbid=YOUR_FACEBOOK_ID

또는 다음과 같이 요청하실 수 있습니다.

POST https://polyglot-inc.com/api/v1/privacy/facebook-delete
Content-Type: application/json

{ "fbid": "YOUR_FACEBOOK_ID" }

※ 실제 엔드포인트/도메인은 귀사 운영 환경에 맞게 구성하십시오.

삭제 대상 및 예외

  • 삭제 대상: 계정 프로필(이름/닉네임, 이메일, 아바타), 소셜 로그인 식별자, 설정 값, 앱 내 생성 콘텐츠(선택: 익명화 가능).
  • 보존/예외: 법적 의무에 따른 결제·세무 서류, 보안 로그, 백업 스냅샷(아래 기한 내 영구 삭제).

처리 기한 및 보존 기간

  • 접수 확인: 영업일 기준 7일 이내
  • 본 삭제 처리: 접수 확정 후 최대 30일 이내
  • 백업·로그 파기: 최대 90일 이내 순차 영구 삭제(기술적 제한 고려)

API 예시 (cURL)

# 표준 계정 삭제 요청
curl -X POST https://polyglot-inc.com/api/v1/account/delete \
  -H "Authorization: Bearer <YOUR_TOKEN>" \
  -H "Content-Type: application/json" \
  -d '{ "reason": "delete-my-account" }'

# Facebook 전용 삭제 요청
curl -X POST https://polyglot-inc.com/api/v1/privacy/facebook-delete \
  -H "Content-Type: application/json" \
  -d '{ "fbid": "1234567890" }'
